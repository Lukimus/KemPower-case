# Prototype scope  

Tämän tuotteen tarkoitus on helpottaa ja nopeuttaa hotellien valojen hallintaa.  
Yritämme ratkaista ongelmaa suurissa rakennuksissa missää monesti valoja unohdetaan päälle tai jätetään turhaan yön yli päälle.  
Tuote luodaan olemaan mahdollisimman muokattava asiakkaan tarpeisiin ja se olisi mahdollisimman helppoa.

1. Tuote pitää saada sammutettua manuaalisesti ja automaattisesti.  
2. Tuote pitää saada sammutettua manuaalisesti tietokoneelta.   
4. Tuoteen automatisointi sensoreita voidaan vaihtaa tarpeen mukaan. 
6. Valoja voidaan ohjelmoida sammumaan tai käynnistymään tiettyyn kellon aikaan.
